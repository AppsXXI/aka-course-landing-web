Módulos generales

Git y GitHub

Git es una --sino la-- herramienta de control de versiones, útil para cualquier área, pero fundamental y necesaria para los programadores.

GitHub es un servidor de repositorios para alojar proyectos de Git.

Desarrollo WEB con HTML y CSS

La WEB entera está escrita en HTML y CSS, donde el HTML representa el significado de su contenido y CSS representa los aspectos visuales de los elementos.

Desarrollo WEB Responsive

El Desarrollo WEB Responsive es la capacidad de un sitio web de adaptarse a diferentes tamaños de pantalla. Querés que tu sitio se vea correctamente en un teléfono? Se Responsive!

Programación con JavaScript

Qué sería de la web sin interacción? El usuario es el objetivo y la interacción es una pieza fundamental para captar su atención.

Programa desde interacciones básicas hasta complejas aplicaciones con JavaScript.

NodeJS y Express

JavaScript ya no sólo del lado del cliente, sino en todos lados. Ahora con NodeJs podemos programar todo, aplicaciones, servidores y mucho más.

Con ExpressJS llevamos Node a otro nivel, programar un servidor web nunca fue más sencillo.
